# Контакты студента

Список доступных полей:

* Почта;
* Телефон;
* СНИЛС студента (СНИЛС будет показан только для студентов - участников по Содействию занятости).

:::info

Просмотр контактов преподавателей и администраторов студентам не доступен.

:::
